Para crear la base de datos SqlServer
	Modificar el string de conexión en "appsettings.json"
	Abrir el Package Manager Console y ejecutar: Update-Database

Se utiliza
	AspNetCore 2.0
	Entity Framework Core
	EF Migrations
	Razor
	Html Tag Helpers
	Authorization (SqlServer)
	Authentication (SqlServer)
	MVVM
	Routing
	Dependency Inyection (NetCore BuiltIn en Controllers)
	Dependency Inyection (NetCore BuiltIn en Views)
